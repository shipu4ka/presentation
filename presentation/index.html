CSS Grid Layout - what is it?

CSS Grid Layout is a two-dimensional layout.

Rick: 'Hello gues! Today we will talk about CSS GRID LAYOUT. WHAT IS IT?' Im: Hey! I'm sorry this can get boring, so I called Rick for help. CSS Grid Layout is a 2D layout. With it, you can control the position of an element vertically and horizontally at the same time, unlike Flexbox, where you can only control elements along one axis.
grid is a set of intersecting horizontal and vertical lines that divide the space of a grid container into grid areas into which grid items can be placed. grid lines are invisible horizontal and vertical dividing lines, they exist on both sides of a row and column. They can be referenced by a numerical index or by a name given in the CSS code. grid track - the space between two adjacent grid lines, used to define either a column or row of a grid. The track goes from one end of the container to the other. By default, adjacent tracks fit tightly to each other, you can set the distance between them using the gap property. grid cell - surrounded by four grid lines. This is the area where content can be placed. This is the smallest grid unit. grid area - a rectangular area bounded by four grid lines and consisting of one or more neighboring cells. The area can be as small as one cell, or as large as all cells in the grid.
